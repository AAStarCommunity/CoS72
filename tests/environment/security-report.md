## ⚠️ 安全建议

### 开发时注意事项
1. **永远不要**将私钥提交到任何版本控制系统
2. **定期轮换**开发和测试用的私钥
3. **使用硬件钱包**处理生产环境私钥
4. **限制访问**环境文件到必要的开发人员

### 生产环境建议
1. **使用多重签名**钱包处理关键操作
2. **启用硬件安全模块**(HSM)保护私钥
3. **实施访问控制**和审计日志
4. **定期备份**和恢复测试
